# Lista de Endpoints da Aplicação

## Usuários (/users)
POST /users/ - Criar usuário
Payload:
{
  "fullName": "Nome Completo",
  "cpf": "12345678900",
  "email": "email@exemplo.com",
  "phone": "999999999",
  "password": "senha",
  "profiles": [1, 2] // IDs dos perfis (ex: 1=ADMIN, 2=APICULTOR, 3=MELIPONICULTOR)
}
GET /users/me - Retorna dados do usuário logado
PATCH /users/{user_id}/activate - Ativar usuário
Payload: (vazio)
PATCH /users/{user_id}/deactivate - Desativar usuário
Payload: (vazio)
PATCH /users/{user_id}/limits - Atualizar limites de apiários/meliponários
Payload:
{
  "max_apiaries": 5,
  "max_meliponaries": 3
}
PATCH /users/{user_id}/roles/add - Adicionar perfil(s) ao usuário
Payload:
{
  "profiles": [1, 2]
}
PATCH /users/{user_id}/roles/remove - Remover perfil(s) do usuário
Payload:
{
  "profiles": [1, 2]
}
GET /users/{user_id}/config - Consultar configuração do usuário
Payload: (vazio)

## Autenticação (/auth)
POST /auth/login - Login
Payload (form):
username: email@exemplo.com
password: senha
POST /auth/recuperar-senha - Recuperação de senha
Payload:
{
  "email": "email@exemplo.com"
}

## Meliponário (/meliponary)
POST /meliponary/ - Criar meliponário
Payload:
{
  "name": "Meu Meliponário",
  "latitude": "-20.123",
  "longitude": "-40.123",
  "tipoInstalacao": "fixo",
  "especieAbelha": "Jataí",
  "quantidadeColmeias": "10",
  "outrosMeliponariosRaio1km": true,
  "qtdColmeiasOutrosMeliponarios": "2",
  "fontesNectarPolen": true,
  "disponibilidadeAgua": true,
  "sombreamentoNatural": true,
  "protecaoVentosFortes": true,
  "distanciaSeguraContaminacao": true,
  "distanciaMinimaConstrucoes": true,
  "distanciaSeguraLavouras": true
}
GET /meliponary/all - Listar todos os meliponários
Payload: (vazio)
GET /meliponary/ - Listar meliponários do usuário
Payload: (vazio)
GET /meliponary/{id} - Detalhar meliponário
Payload: (vazio)
PUT /meliponary/{id} - Atualizar meliponário
Payload: igual ao de criação
DELETE /meliponary/{id} - Deletar meliponário
Payload: (vazio)

  "distanciaMinimaConstrucoes": true,

GET /apiary/all - Listar todos os apiários
Payload: (vazio)
GET /apiary/ - Listar apiários do usuário

## Mapas (/maps)
POST /maps/upload/ - Upload de mapa
Payload: arquivo (multipart/form-data)
GET /maps/ - Listar mapas
Payload: (vazio)
GET /maps/content/{filename} - Obter conteúdo do mapa
Payload: (vazio)
DELETE /maps/{map_id} - Deletar mapa
Payload: (vazio)
